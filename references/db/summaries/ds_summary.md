# 数据结构模块摘要

## 核心主题

本模块介绍算法竞赛中常用的数据结构，包括线性结构、树形结构和高级数据结构。

## 主要内容

### 1. 线性数据结构
- 数组与字符串
- 栈和队列
- 链表

### 2. 树形数据结构
- 二叉树遍历
- 二叉搜索树
- AVL树
- 红黑树
- Treap/Splay

### 3. 并查集 (DSU)
- 基本操作
- 路径压缩
- 按秩合并
- 扩展应用

### 4. 堆
- 二叉堆
- 左偏树
- 配对堆

### 5. 树状数组 (Fenwick Tree)
- 单点更新
- 区间查询
- 二维树状数组

### 6. 线段树 (Segment Tree)
- 区间更新
- 区间查询
- 懒标记
- 主席树

### 7. 高级数据结构
- 平衡树
- 分块
- 块状链表
- 可持久化数据结构

## 学习建议

1. **入门阶段**：掌握数组、栈、队列、链表
2. **提高阶段**：学习并查集、堆、树状数组
3. **进阶阶段**：掌握线段树和高级数据结构

## 典型例题

- 并查集：连通性问题、最小生成树
- 树状数组：逆序对、区间和
- 线段树：区间最值、区间更新

## 重要概念

- 时间复杂度
- 空间复杂度
- 操作次数
- 常数优化
