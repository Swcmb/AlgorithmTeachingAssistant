# 字符串模块摘要

## 核心主题

本模块介绍字符串处理算法，包括字符串匹配、哈希、后缀自动机等。

## 主要内容

### 1. 字符串基础
- 字符串表示
- 子串操作
- 字符串比较

### 2. 字符串哈希
- 多项式哈希
- 双哈希
- 前缀哈希

### 3. KMP算法
- 模式匹配
- 前缀函数
- 应用扩展

### 4. Trie树
- 字典树
- 前缀匹配
- 异或问题

### 5. AC自动机
- 多模式匹配
- 失配指针
- 文本扫描

### 6. 后缀数组 (SA)
- 后缀排序
- LCP数组
- 应用

### 7. 后缀自动机 (SAM)
- 状态表示
- 转移构造
- 应用

### 8. Manacher算法
- 最长回文子串
- O(n)复杂度

### 9. Z函数
- Z数组计算
- 字符串匹配

## 学习建议

1. **入门阶段**：掌握字符串哈希和KMP
2. **提高阶段**：学习Trie和AC自动机
3. **进阶阶段**：掌握后缀数组和后缀自动机

## 典型例题

- 字符串匹配：KMP、Z函数
- 回文问题：Manacher算法
- 后缀处理：SA、SAM

## 重要概念

- 时间复杂度
- 空间复杂度
- 匹配效率
